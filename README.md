# 🏐 Sorteio & Balanceamento de Times (ELO)

Um site simples para **sortear e balancear times de vôlei ou futebol**, levando em conta:
- ELO de cada jogador
- Quantidade mínima e máxima de jogadores por time
- Placar mínimo e máximo
- Registro do histórico de peladas com data e hora

## 🚀 Como usar
1. Baixe o repositório:
   ```bash
   git clone https://github.com/SEU-USUARIO/sorteio-times.git
   cd sorteio-times
   ```
2. Abra o arquivo:
   ```bash
   index.html
   ```

## 🌐 Publicação com GitHub Pages
1. Vá em Settings → Pages.
2. Escolha:
   - Branch: main
   - Folder: / (root)
3. Clique em Save
4. Acesse: `https://SEU-USUARIO.github.io/sorteio-times/`
